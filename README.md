# python skelly

python skelly cumulation of common and novel methods to distribute so called installable python packages.

```bash
[ -d $HOME/python_skelly ]
$ cd python_skelly
$ zip -r ../python_skelly.zip *
$ cd ..
python python_skelly.zip

```
run
```bash
python3 python_skelly.zip
```
